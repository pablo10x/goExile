This build only supports headless mode.

INSTRUCTIONS
To start Exile in headless mode, run:
	./ExileHeadless_DedicatedServer.sh
or run:
	./DedicatedServer_Core.x86_64 -batchmode -nographics

You might have to grant execute permissions first by running:
	chmod +x ExileHeadless_DedicatedServer.sh
	chmod +x DedicatedServer_Core.x86_64
